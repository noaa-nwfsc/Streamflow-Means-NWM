NOAA Water Resource Region 18 (WR18) Streamflow Data

hydrology streamflow NOAA Zarr NetCDF open data

Description

This dataset contains daily mean streamflow data derived from the hourly output of the NOAA National Water Model v3.0 for Water Resource Region 18 (WR18), which includes most of California and portions of Nevada.

The data has been spatially subset using NHDPlusV2.1 flowlines (COMIDs) and temporally aggregated from hourly to daily means.
